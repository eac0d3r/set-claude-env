# set-claude-env

CLI tool to set environment variables for Claude Code.

This tool is designed to help you set up the necessary environment variables for using Claude Code in your development environment. It simplifies the process of configuring your system to work with Claude Code by providing a command-line interface to set model variables.

## Installation

To install the `set-claude-env` tool, you can use npm. Run the following command in your terminal:

```bash
npm install -g set-claude-env
```

Or run with npx:

```bash
npx set-claude-env
```
